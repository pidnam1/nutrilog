About
Nutrilog is a web application designed to help people without access to nutritional value information find healthy food options. Built using HTML, CSS, Python, Django, and Heroku, Nutrilog harnesses Google's Cloud Vision API as well as the Nutritionix Database to provide instant nutritional feedback. Users can upload a picture of a handwritten grocery list or type one in manually to recieve the nutritional information for each food on the list, as well as recommendations for healthier options.
Built for HooHacks 2021
by Mandip Bhadra, Wahaj Syed, Ana Borges, and Spencer Jackson
